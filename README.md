# factory_task
Задание для вакансии

# windows 10 - powershell - Visual Studio Comunity 17.32
```sh
git clone git@github.com:alec-chicherini/factory_task.git
cd .\factory_task\
mkdir build
cd build
cmake ..
cmake --build .
cd debug
./factory_task.exe
echo done
```
